# SSRD

SSRD (Self-Service Request Dashboard) is our central platform for managing modules. It provides the tooling needed to streamline module registration, release management, and related services.

- [Registering Modules](/docs/modules/ssrd/registering-modules)
- [Additional Services](/docs/modules/ssrd/additional-services)
