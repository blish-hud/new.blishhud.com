:::caution tl;dr
Blish HUD has tens of thousands of daily users, resulting in thousands of module downloads.
Keep those users in mind when publishing changes that may inadvertently affect them.
By publishing through the public repo you agree to follow the recommendations in these module citizen guides.
:::

Blish HUD can run with any combination of modules on a wide range of systems and configurations, so each module must behave as reliably as it can.
