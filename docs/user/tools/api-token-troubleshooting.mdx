## Guild Wars 2 API Stability

The Guild Wars 2 API has been known to experience long periods of instability.  You can check any of the following status pages for more information:
- [Gw2Efficiency API Status](https://status.gw2efficiency.com/)
- [gw2treasures.com API Status](https://en.gw2treasures.com/status/api?period=24h)
- [Aleeva API Status](https://status-api.aleeva.io/)
