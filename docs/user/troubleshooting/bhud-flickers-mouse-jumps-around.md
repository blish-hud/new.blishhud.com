title: Blish HUD flickers and the mouse jumps around
---

If you're using an AMD card, please ensure that "AMD Fluid Motion Frames" is disabled.

![image](https://github.com/user-attachments/assets/26ab4a99-4e0a-4f32-b5d3-088ce3b9051c)

While on, this setting seems to cause the flickering issue for many overlays, including Blish HUD.

It will also make the mouse move on its own.
