## Issue

While running Blish HUD on an AMD GPU, game performance becomes choppy/FPS is impacted more than expected.

## Solution

Ensure these AMD GPU settings are disabled for Blish HUD.
- Radeon Enhanced Sync
- AMD Fluid Motion Frames
- Radeon Chill
- GPU Scaling / Radeon Super Resolution
- Radeon Image Sharpening

If none of those changes help, please also review our general [performance troubleshooting guide](https://blishhud.com/docs/user/troubleshooting/performance-troubleshooting).
